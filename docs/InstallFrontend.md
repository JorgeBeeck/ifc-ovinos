## Requisitos

---

- [Node.js](https://nodejs.org/)

```sh
# Faça as operações apartir deste diretorio
cd frontend

# Instalar as dependências
npm install
# Executar em modo de desenvolvimento
npm run dev
```

O frontend responde localmente na porta 3000: http://localhost:3000/.
